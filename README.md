# Rebalancing Vault Calculator
Rebalancing Vault Calculator is a tool that calculates the most profitable APY allocation across multiple DeFi strategies. It analyzes parameters such as LTV, liquidation LTV, and strategy-specific APYs to determine the optimal distribution of capital (e.g., USDC deposits) across different strategies.

Given an input amount (e.g., $1,000), the calculator evaluates risk constraints and yield mechanics to compute the maximum achievable APY and provides an optimal rebalancing breakdown to maximize returns while staying within safe liquidation thresholds.
