# API Testing Examples for Fund Calculator

URL: http://localhost:5000/api/max-profits
Method: GET

## 2 Funds Example

Body (raw JSON):
{
    "total_my_supply": 100000,
    "fee_percentage": 0.08,
    "total_borrowed1": 179954,
    "total_supplied1": 370406,
    "optimal_usage_ratio1": 0.85,
    "variable_rate_slope1_1": 0.08,
    "variable_rate_slope2_1": 0.8,
    "token_price1": 1,
    "total_borrowed2": 522821,
    "total_supplied2": 1792518,
    "optimal_usage_ratio2": 0.75,
    "variable_rate_slope1_2": 0.11,
    "variable_rate_slope2_2": 0.7,
    "token_price2": 1
}

Expected Response Format:
{
    "fund1_supply": [number],
    "fund2_supply": [number],
    "total_profit": [number]
}

Method: GET

Body (raw JSON):

    "fee_percentage": 0.08,
    "total_borrowed1": 179954,
    "total_supplied1": 370406,
    "optimal_usage_ratio1": 0.85,
    "variable_rate_slope1_1": 0.08,
    "variable_rate_slope2_1": 0.8,

    "total_borrowed2": 522821,

    "optimal_usage_ratio2": 0.75,
    "variable_rate_slope1_2": 0.11,
    "variable_rate_slope2_2": 0.7,
